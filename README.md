# psycologyTests-app
Psychology tests app build in React Native with Expo

# setup enviroment
  1- Clone the repo with "git clone git@github.com:TF-ING-INF-UCA-2022/psycologyTests-app.git"
  2- Install node v18.12.1
      * if you have nvm run:
      * nvm install 18.12.1
      * nvm use 18.12.1
  3- Run "npm i"
  4- Start the app with "npm run start"
